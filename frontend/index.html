About This Slice

Enhanced Mobile Broadband (eMBB) is designed to support high data rate applications with improved user experience. This slice handles bandwidth-intensive services such as 4K/8K video streaming, virtual reality (VR), augmented reality (AR), and cloud gaming. eMBB focuses on delivering peak data rates up to 20 Gbps with consistent quality of experience across diverse environments.

Ultra-Reliable Low Latency Communications (URLLC) provides mission-critical communication services with stringent requirements for latency and reliability. This slice supports applications like autonomous vehicles, remote surgery, industrial automation, and smart grid control. URLLC targets end-to-end latency as low as 1ms with 99.999% reliability.

Massive Machine Type Communications (mMTC) enables connectivity for a massive number of IoT devices with low power consumption and extended coverage. This slice supports smart cities, agricultural sensors, asset tracking, and industrial IoT deployments. mMTC can handle up to 1 million devices per square kilometer with optimized energy efficiency.

Available Models

Comprehensive architecture information with download and documentation access

VAR+GRU+TFT

Transformer
Overview

Combines Vector Autoregression (for linear relationships) and GRU (for short-term temporal patterns) with a Temporal Fusion Transformer.

Use Case Fit

Highly interpretable. It allows us to understand which specific network features caused a predicted spike in traffic, making it ideal for multi-horizon forecasting where we need to explain network behavior to stakeholders.

View PDF

VAR+GRU+N-BEATS

MLP-based
Overview

Uses Neural Basis Expansion Analysis for pure deep learning time-series modeling.

Use Case Fit

Excels at trend and seasonality decomposition. 5G traffic has very strong daily and weekly seasonal patterns; this model automatically isolates these repeating patterns to predict future network loads accurately.

View PDF

VAR+GRU+PatchTST

Transformer
Overview

A patch-based Transformer that divides time-series data into sub-patch tokens, maintaining channel independence.

Use Case Fit

Perfect for long-range historical data. By looking at 'patches' of time rather than individual moments, it captures local semantic information efficiently, allowing us to process massive amounts of past network metrics without running out of memory.

View PDF

VAR+GRU+TimeMixer

Hybrid
Overview

Utilizes a multi-scale decomposition approach, mixing both past historical data and future feature predictions.

Use Case Fit

The best all-rounder for volatile traffic. It simultaneously analyzes micro-scale events (sudden millisecond bursts) and macro-scale trends (overall daily data usage), making it highly adaptive to sudden changes in 5G network demand.
